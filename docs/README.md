#기능
* 사용자의 숫자입력 기능 구현
    * 입력값 검증
* 컴퓨터에서 숫자 생성
    * 생성된 숫자 검증
        * 자리수가 정상적으로 생성되었는지 검증
        * 같은 숫자가 있는지 확인하여 있다면, 재생성로직 수행
* 결과 검증
    * 컴퓨터에서 생성된 숫자가 입력한 숫자에 있는지 확인하여 위치 확인
    * 숫자가 아예 없다면 낫싱처리
    * 숫자가 있고, 위치까지 정확한 개수만큼 스트라이크 처리
    * 숫자가 있지만, 위치까지 정확하지 않다면 볼 처리

* 3스트라이크 시 게임 재시작 여부 문의
    * 1 입력 시 재시작
    * 2 입력 시 게임종료
